# Chartjs

## Sample
http://programmerbox.com/2013-05-24_chart.js_tutorial/

## Setup
1. Chart.jsをダウンロードする
2. Chart.jsを「vendor/assets/javascripts」に配置する
3. Chart.jsを「app/assets/javascripts/application.js」でrequireする
```
//= require Chart.min.js
```
